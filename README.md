# 🚀 Proyecto de Desarrollo de Software Web  
### Backend • Frontend • Base de Datos • Docker

Este repositorio contiene el desarrollo de una aplicación web completa que integra **frontend**, **backend**, **base de datos** y un entorno de despliegue mediante **Docker**. El objetivo es construir una solución modular, escalable y fácil de ejecutar en cualquier entorno.

---

## 📌 Descripción General

El proyecto está organizado en tres componentes principales:

- **Frontend:** Interfaz gráfica del usuario.  
- **Backend:** API o servidor encargado de la lógica del negocio.  
- **Base de Datos:** Sistema de persistencia de la información.  

Todo el ecosistema se orquesta mediante **Docker Compose**, permitiendo levantar la aplicación completa con un solo comando.

---

## 🧱 Estructura del Proyecto

```plaintext
/project-root
│

├── backend/             # API REST o servidor de aplicación
├── database/            # Scripts, seeds o configuraciones de BD
```
